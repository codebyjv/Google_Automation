# 📌 Automação de Processos com Google Apps Script

## 💡 Problema
Processos internos manuais demandavam tempo elevado para envio de e-mails e organização de dados.

## 🚀 Solução
Desenvolvi uma automação utilizando Google Apps Script integrada com HTML, permitindo:
- Envio automatizado de e-mails
- Integração com Google Sheets
- Interface simples para operação

## 🛠 Tecnologias
- Google Apps Script (JavaScript)
- HTML / CSS
- Google Sheets API

## 📈 Resultado
Redução significativa de tarefas manuais e aumento da eficiência operacional.

## ⚠️ Observação
Algumas informações sensíveis foram removidas por segurança.
